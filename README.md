# Cosmos Empires Calculator

A small web app to calculate scores for the engine-building card game "Cosmos Empires".

Live demo
- GitHub Pages: https://ra314.github.io/cosmos-empires-calculator/

About the game
- BoardGameGeek: https://boardgamegeek.com/boardgame/365211/cosmos-empires

Usage
- Open index.html in your browser or visit the GitHub Pages link above.
- The app is a static HTML site (no build step required).
